== Description ==

Essa é a versão gratuita do Super Links, que permite a criação de links usando seu próprio nome de domínio que redirecionam para seus links de afiliado. Coloque seus links em categorias que irão organizar
e facilitar muito o trabalho com seu site/blog.

= Super Links =

O Super Links permite que você crie links usando seu próprio nome de domínio que redirecionam para seus links de afiliado de forma camuflada ou não. Você pode inserir códigos de rastreio do Facebook e Google de forma simples e fácil. E ainda monitorar a quantidade acessos que cada link obteve.
Além disso, ainda é possível camuflar seus links para que usuários de determinados países sejam redirecionados para seus links de afiliados, enquanto robôs e outros acessos (Facebook e Google por exemplo) visualizem outra página que você configurar.

= Funcionalidades da versão Light =

* Crie links encurtados usando seu próprio nome de domínio que redirecionam para qualquer outra url
* Execute testes A/B, criando links que redirecionam para até 3 url's diferentes e que vão alternando de acordo com os acessos recebidos
* Habilite ou desabilite um link de acordo com a sua necessidade
* Organize seus links em categorias
* Tenha uma visualização rápida, simples e de fácil entendimento para criar e analisar as métricas dos seus links
* Analise as métricas de totais de acessos por link
* Analise as métricas de acessos únicos por link
* Faça importação dos seus links criados com outros plugins

== Installation ==

1. Envie o arquivo 'super-links-light.zip' para o diretório '/wp-content/plugins/'

2. Ative o plugin através do menu 'Plugins' em seu painel administrativo do WordPress

3. Certifique-se de ter alterado as Configurações de links permanentes no menu 'Configurações' do seu painel WordPress para qualquer configuração diferente da padrão
