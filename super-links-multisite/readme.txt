Plugin de encurtamento de links otimizado para WordPress Multisite configurado com subdomínios.

== Description ==

O Super Links Multisite é uma versão modificada do plugin Super Links Light, especialmente adaptada para funcionar com WordPress Multisite sem sistema de ativação obrigatório.

**Principais Características:**

* ✅ Encurtador de links (Redirecionamento PHP)
* ✅ Encurtador de links para Whatsapp e Telegram  
* ✅ Categorização de links
* ✅ Testes A/B
* ✅ Importar links de outros plugins
* ✅ Compatível com WordPress Multisite
* ✅ Suporte a subdomínios
* ✅ Sem sistema de ativação obrigatório

**O que foi modificado:**

1. **Removido Sistema de Ativação**: Eliminado completamente o sistema de ativação por telefone
2. **Adaptado para WordPress Multisite**: Otimizado para funcionar com configuração de subdomínios
3. **Interface Simplificada**: Nova tela inicial focada nas funcionalidades do plugin
4. **Código Limpo**: Removidos métodos desnecessários relacionados à ativação

== Installation ==

1. Faça o upload da pasta `super-links-multisite` para o diretório `/wp-content/plugins/`
2. Ative o plugin na rede (Network Admin > Plugins)
3. Configure o plugin conforme necessário
4. Comece a criar seus links encurtados

== Frequently Asked Questions ==

= O plugin funciona com WordPress Multisite? =
Sim! Esta versão foi especialmente adaptada para WordPress Multisite com configuração de subdomínios.

= Preciso ativar o plugin com telefone? =
Não! Removemos completamente o sistema de ativação. O plugin está sempre pronto para uso.

= Posso importar links de outros plugins? =
Sim, o plugin mantém a funcionalidade de importação de links de outros plugins similares.
